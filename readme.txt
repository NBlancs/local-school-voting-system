How to run

1. Download the Zip File named "votesystem"

2. Put the file sa htdocs na file in the xampp folder in local disk C: (sulod na sya sa xampp sa imong C: nga drive)

3. Open Xampp and start Apache and MySQL 

4. Open PhpMyAdmin (http://localhost/phpmyadmin). Click the import option. After that, click Choose file. go to the votesystem folder. Find a folder named "db" then click that and import. Finally, click the import button naa sya sa ubos (just scroll down).

5. Run the script. (Go to your choosen browser(Google, Edge, Firefox))
http://localhost/votesystem/admin/index.php
